# Jasvinder Wick: The Game

A static browser game ready to publish with GitHub Pages.

## Project structure

```text
index.html
assets/
  audio/
  images/
  screenshots/
```

The game uses relative asset paths, so it works when hosted from a GitHub Pages
project URL such as `https://samin-khan.github.io/jasvinder-wick-the-game/`.

## Publish with GitHub Pages

1. Push the repo to GitHub.
2. Open the repository on GitHub.
3. Go to **Settings > Pages**.
4. Set **Source** to **Deploy from a branch**.
5. Select branch `main` and folder `/root`.
6. Save.

GitHub Pages will publish the game from `index.html`.
